Cloudoll, 云时代的微服务玩具

开始 Cloudoll

当前版本： v0.2.2 | 文档更新于：2015-8-25

帮助您快速构建分布式的微服务应用

cloudoll-server

注册中心，管理所有的微服务，并能在web端实现实时监控和报警。

KoaApplication

自动的 koajs 应用程序，只需简单的配置，则可以得到一个强大的『配置优先的』Web 应用程序。

自动路由

自动映射开放接口，管理接口和内网接口。当然你也可以通过修改配置文件方便的修改路由目录。

数据快速访问

支持 mongo、mysql、postgres、redis 等数据库的快捷访问，一个方法搞定查询。

权限验证

自动提供内网接口的权限验证。依赖 cloudarling 微服务，实现管理接口的权限验证

错误统一处理

只需简单 throw，在前台页面显示友好的错误信息，亦可以方便的在控制台查看错误详情。

controller 基类

已经提供了基础的 service 和 controller，通过继承，可以无需编写代码实现后台管理接口。

自动的数据校验

通过编写 json schema，实现在数据提交时候的自动验证。

远程微服务调用

通过工具的封装，让远程的微服务调用就像在调用本地方法一样。

更多 Cloudoll 相关的开源项目

cloudarling

官方提供的帐号管理系统（passport）微服务。

cloudoll-ui

快速创建前端管理界面，当然他也可以用不依赖 cloudoll。

cloudoll-java

基于 sparkjava，快速构建适配 cloudoll 的微服务。

cloudoll-android

android 版本微服务管理和远程调用工具。

Cloudoll 是完全开源的。他目前托管在 GitHub。
